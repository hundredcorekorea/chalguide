## 1. 전체 화면 맵 (최종)

```
[첫 방문]
    │
    ├─ "숙련자" 선택 ──→ [홈 대시보드]
    │
    └─ "10년 만에 복귀" 선택 ──→ [해동 가이드]
                                    │
                                    └─ 언제든 홈으로 이동 가능

[홈 대시보드]
    │
    ├── 핵심 기능 (하단 네비 5개)
    │   ├── /score      점수 계산기 + 티어 로드맵
    │   ├── /boss       보스 체크리스트 + 결정석 시뮬
    │   ├── /quest      주간퀘스트 가이드
    │   ├── /jobs       직업 추천 (47개 전 직업)
    │   └── /hunting    사냥터 가이드

    │   ├── /defrost    냉동용사 해동 가이드
    │   ├── /sim        시뮬레이터 모음
    │   │   ├── 의문의 결계 시뮬
    │   │   ├── 스타포스 기대비용
    │   │   ├── 비약 최적화
    │   │   └── 경험치 계산기
    │   ├── /core       5차/6차 코어 가이드 (직업별)

    └── Phase 2 (트래픽 검증 후)

        ├── /patch      패치노트 AI 요약
        └── /routes     루트 공유 (유저 참여)
```

## 2. 하단 네비게이션 (모바일 5탭)

```
┌──────────────────────────────────────┐
│  🏠     🧮     ⚔️     📋     🎮    │
│  홈    점수   보스   퀘스트  직업    │
└──────────────────────────────────────┘
```

- 5개만. 사냥터/시뮬/코어는 홈에서 카드로 진입
- 데스크톱은 상단 네비에 전체 메뉴 표시

**왜 이 5개인가:**
- 홈: 진입점. 초기화 카운트다운 + 기능 카드
- 점수: SEO 킬러 ("챌섭 점수 계산" 검색 유입)
- 보스: 매주 반복 방문 (LTV 핵심)
- 퀘스트: 매주 반복 방문 (LTV 핵심)
- 직업: 냉동용사 첫 질문 ("뭐 키울까?")

## 4. 유저 플로우 (핵심 시나리오)

### A: 냉동용사 — "10년 만에 돌아옴"

```
구글 "메이플 복귀 가이드" → 웹 유입
→ "10년 만에 복귀" 클릭 → 해동 Step 1~6
→ (각 스텝에서 은어 팝업 활용)
→ Step 6 완료 → 홈 전환
→ 직업 추천 → "뭐 키울까?" → 렌/보마 선택
→ 점수 계산기 → "에메 언제 가?"
→ 매주 보스 체크하러 재방문
```

### B: 점수 유저 — "에메 언제 가?"

```
구글 "챌섭 점수 계산" → 웹 유입
→ 점수 계산기 → 결과 확인
→ 보스 체크리스트 → 이번 주 보스 체크
→ 주간퀘스트 → 미션 체크
→ 매주 목요일마다 재방문
```

### C: 숙련자 — "보스 체크하러 왔다"

```
앱 열기 → 보스 체크리스트
→ 체크 → 포인트 확인 → 이탈
→ 1~2분, 주 1~2회 반복
```
