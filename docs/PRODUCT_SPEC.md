## 1. 제품 개요

### 컨셉

**"챌린저스 할 때 = 챌섭가이드"**

환산주스텟이 "환산 계산할 때 가는 곳"이고 츄츄지지가 "시뮬레이션할 때 가는 곳"이듯, 챌섭가이드는 **"챌린저스 월드 시작하면 제일 먼저 여는 곳"**이 되어야 한다.

기능 하나가 뾰족한 게 아니라, **챌린저스라는 카테고리 자체를 먹는다.** 점수 계산, 보스 체크, 사냥터, 해동 가이드 — 이 모든 게 "챌린저스의 전체 여정"이고, 첫 화면에서 "지금 내 상황에 맞는 다음 할 일"을 바로 보여준다.

### 포지셔닝

- **"챌린저스 = 챌섭가이드"** — 카테고리 자체를 선점
- 환산주스텟/츄츄지지처럼 이름이 곧 포지셔닝. "챌섭가이드"를 검색하면 끝
- 기능이 여러 개여도 진입점은 하나 — "지금 뭐 해야 해?" 에 답하는 원스톱
- 웹(SEO) + 앱(Play Store) 동시 노출
- 시즌 오픈 2주 전 출시 → 시즌 종료 후 방치 → 다음 시즌 전 데이터만 교체
- **리부트북과 완전 별개.** 챌린저스는 유니온/링크 없음, 정보 범위 다름. 주간퀘스트 UX 패턴만 참고.

## 2. 타겟 유저

### MVP 핵심 타겟 (이 두 놈만 판다)

| 유저 유형 | 니즈 | MVP 대응 기능 |
|-----------|------|--------------|
| **냉동용사 (10년+복귀)** | 아케인포스가 뭔지도 모름. 시스템 자체가 외국어 | **해동 가이드** + Glossary |
| **점수 달성 목표 유저** | "에메 가려면 얼마나 걸려?" 매주 보스/퀘스트 체크하러 옴 | **점수 계산기** + 보스/퀘스트 체크리스트 |

> MVP는 결핍이 가장 심한 유저 하나만 패는 것. 이것저것 다 담으면 김밥천국 된다.
> 냉동용사 = SEO 유입 ("메이플 복귀 가이드"), 점수 유저 = 반복 방문 (LTV).
> 이 두 축이 트래픽과 수익을 만든다.

### 확장 타겟 (Phase 2 이후, 트래픽 검증 후)

| 유저 유형 | 니즈 |
|-----------|------|
| 최소 노력(주차) 유저 | 사냥 없이 비약/이벤트만으로 275 달성 로드맵 |
| 보스 도전 유저 | 보스 체크리스트, 드롭 정리 |
| 파트너 유기 피해 유저 | 단독 미션 점수 복구 가이드 |
| 공략 영상 시간 없는 유저 | 핵심 정보만 빠르게 (트렌드 카드) |

## 4. 핵심 기능 (MVP)

### 4-1. 점수 계산기 & 로드맵 가이드 ✅ 구현 완료 + v2.2 업데이트 예정

- 현재 점수 입력 → 목표 등급까지 필요 점수 계산
- 8단계 티어 전체 표시 (비기너~챌린저)
- 일일 획득 가능 점수 기반 D-day 예측
- 프로그레스 바 시각화
- 티어별 보상(결계 해제 크기 등) 목록 + 달성 여부 표시
- **[v2.2] 무지성 4만 점(다이아) 로드맵**: 사냥/보스/레벨업 미션을 조합한 최단기 달성 체크리스트
- **[v2.2] 성장의 비약 스케줄러**: 챌린저스 패스 및 이벤트로 얻는 비약(초성비, 궁극의 유니온 성장 비약 등) 최적 사용 구간 계산 (버닝 비욘드 1+1 구간 적용)

### 4-2. 직업 추천 ✅ 구현 완료

- 플레이스타일 기반 직업 추천 (사냥형/보스형/유틸형)
- S/A/B 티어 분류
- 직업별 장점 요약
- 스타일 필터 토글

### 4-3. 사냥터 가이드 ✅ 구현 완료

- 12개 사냥 구간 (검증된 데이터 기준)
- 레벨 입력 시 해당 구간 하이라이트
- 사냥터별 필요 포스 (아케인포스/어센틱포스) 표시
- 지역 구분 (버닝 이벤트 / 아케인 리버 / 그란디스)

### 4-4. 보스 체크리스트 ✅ 구현 완료

- 26개 보스 (일일/주간/월간 필터)
- 난이도별 체크 (이지/노멀/하드/카오스/익스트림)
- 처치 포인트 자동 합산
- **매주 목요일 자정 자동 초기화** (localStorage)
- 결정석 판매 한도 정보 (캐릭터 12개, 월드 90개)
- 보스 티어 색상 표시 (납/동/은/금)

### 4-5. 주간퀘스트 가이드 ✅ 구현 완료

> 리부트북의 QuestHelperActivity 체크리스트 UX 패턴을 웹 버전으로 리팩토링

- **주간 미션 체크리스트**: 사냥/레벨/보스 미션 전체 목록, 체크 시 localStorage 저장
- **포인트 합산 대시보드**: 완료 미션 수 + 획득 포인트 총합 실시간 계산
- **진행도 프로그레스바**: 주간 완료율 시각화
- **효율 순서 추천**: 포인트/시간 효율 기준 상위 5개 "시간 없을 때 이것만!"
- **자동 초기화**: 매주 목요일 자정 감지 → 체크 상태 리셋
- 보스 미션 포인트 규칙 카드 (이지 100 / 노멀 500 / 하드·카오스 1,000)

### 4-6. 인터랙티브 시뮬레이터 (재미 요소)

유저가 만지작거리면서 시간 쓰는 콘텐츠 = 체류 시간 + 재방문.

#### a) 의문의 결계 시뮬레이터

- 현재 티어 선택 → 해제 가능한 결계 크기(초소형~초대형) 표시
- 결계 오픈 시 기대 보상 목록 + 확률 테이블
- "이 티어에서 결계 몇 개 열 수 있어?" 시뮬레이션
- 결계봉 사용 전략 가이드 (어떤 크기를 먼저 열지)

#### b) 스타포스 기대비용 계산기

- 현재 성 → 목표 성 입력
- 파괴 확률, 기대 메소, 예상 시도 횟수 계산
- 반빨별(파괴 방지) 사용 여부 토글 → 비용 비교
- "12성 → 17성까지 기대비용: 약 X메소 (파괴 Y회 예상)"

#### c) 결정석 수익 시뮬레이터

- 처치 가능한 보스 목록 체크 (보스 체크리스트 연동)
- 주간 결정석 예상 수입 자동 계산
- 파티원 수에 따른 1/n 분배 시뮬레이션

#### d) 비약 최적화 계산기

- 보유 비약 종류·개수 입력, 현재 레벨 입력
- 각 비약의 최적 사용 레벨 구간 표시
- "지금 쓰면 X만큼 손해" / "Y레벨에서 쓰면 최적" 판정

#### e) 경험치 계산기 (목표 레벨까지 소요일)

냉동용사 핵심 질문: **"275까지 며칠이야?"**

- 입력: 현재 레벨, 목표 레벨, 하루 사냥 시간
- 버닝 옵션 토글: 하이퍼버닝(200~260), 버닝 비욘드(260~270)
- 일일 루틴 체크: 몬스터파크, 아케인/그란디스 일퀘, 에픽던전
- 출력: 총 필요 경험치, 예상 소요일, 레벨별 상세 내역
- 챌린저스 패시브 경험치 버프(+50%) 자동 반영

#### f) 직업 추천 (챌린저스 메타 기준) — 필수 기능, 최고 민감도

> 냉동용사가 제일 먼저 하는 고민: **"뭐 키울지 모르겠는데"**
> 이것만으로 유입되는 SEO 키워드: "챌린저스 직업 추천", "챌섭 뭐 키울까"
> **직업 추천이 틀리면 유저가 시즌 내내 고통 → 별점 1점 + 신뢰 붕괴. 가장 민감한 데이터.**

**5소스 룰 (직업 전용 — 일반 공략의 3소스보다 엄격):**

| 조건 | 처리 |
|------|------|
| 5개+ 소스가 동일 평가 | **채택** — 별점 확정 |
| 3~4개 소스 | **참고** — 별점에 반영하되 "의견 갈림" 표시 |
| 2개 이하 | **제외** — 신뢰도 부족 |

소스 = 유튜브 공략 영상 + 인벤/에펨/아카 커뮤니티 글 + 츄츄지지 통계

- 별점 5점 체계: 사냥/보스/난이도/뉴비추천 각각 평가
- 뉴비용 한마디(`newbieTip`) + 뉴비 경고(`newbieWarning`) 포함
- **챌린저스 전용 메타** 반영 (유니온/링크 없으니 자체 성능이 핵심)

```

[넥슨 공홈 패치노트 크롤링] → [Claude Sonnet 요약] → [자동 검증] → [자동 배포]
```

패치노트는 공식 텍스트 기반이라 할루시네이션 리스크 극히 낮음 → 검수 없이 자동 배포.

### 5-3. Phase 3 — 닉네임 조회 (maple.ai.kr 방식 차용)

> Phase 2에서 Nexon API 연동 안정화된 후에만 진행.

**핵심 기능: 닉네임 입력 → 내 데이터 자동 채우기**

```
[유저가 닉네임 입력]
        ↓
[서버에서 Nexon API 호출] (프론트 직접 호출 X — CORS + API 키 보호)
  - /character/{ouid}/stat → 전투력, 레벨
  - /character/{ouid}/item-equipment → 스타포스 현황
  - /character/{ouid}/symbol-equipment → 아케인/어센틱포스
        ↓
[점수 계산기에 자동 입력]
  - 현재 레벨 → 자동 채움
  - 전투력 → 티어별 컷과 비교 ("에메랄드까지 전투력 200만 부족")
  - 포스 현황 → 사냥터 가이드에 "지금 갈 수 있는 곳" 하이라이트
```

**maple.ai.kr과의 차이:**

| maple.ai.kr | 챌섭가이드 |
|-------------|-----------|
| 장비 진단 (AI가 "이 부위 약해") | 수치 자동 채우기 (계산기/가이드에 연동) |
| 모든 서버 지원 | 챌린저스 전용 |
| 실시간 서버 필요 | 조회 결과 localStorage 캐싱 (서버 부하 최소) |
| 메인 기능 | 보조 기능 (편의성 향상) |

**구현 제약:**
- 서버 사이드 API 프록시 필요 (Vercel Serverless Function 또는 별도 API)
- Nexon API Rate Limit 관리 (유저당 조회 횟수 제한)
- **서버비 $0 원칙 깨질 수 있음** → Vercel Serverless 무료 티어(월 10만 회)로 커버 가능한지 Phase 2에서 검증

### 5-7. 데이터 소스 우선순위 (신뢰도 순)

| 순위 | 소스 | 용도 | 성격 |
|------|------|------|------|
| 1 | **넥슨 공식 홈페이지** | 티어 점수, 시즌 일정, 패치노트 | 공식 팩트 |
| 2 | **메이플지지/츄츄지지 API 실측** | 직업 점유율, 전투력 통계 | 316만건 표본 실데이터 |
| 3 | **maple.ai.kr 정리 데이터** | 보스별 포인트, 계산기 수치 | 공식에서 텍스트로 안 주는 것들 |
| 4 | **정심심 블로그** | 직업별 코강/하이퍼/어빌 | 밸패마다 업데이트, 전 직업 커버 예정 |
| 5 | **커뮤니티 5소스 교차** | 직업 추천, 사냥터 효율 | 주관적 의견 교차 검증 |
| 6 | **유튜브 영상 종합** | 공략 트렌드 | 수동 리서치 or Sonnet 자동화 |

**원칙:**
- 상위 소스와 하위 소스가 충돌하면 상위 소스를 우선
- 직업 추천은 2번(실측 점유율) + 5번(5소스 교차)을 조합 — 점유율이 높으면 뉴비가 많이 잡는다는 팩트
- 코어/하이퍼/어빌은 4번(정심심) 기반 + 챌린저스 맞춤 재가공
- 보스 포인트는 3번(maple.ai.kr) 인용 — 공식에서 텍스트로 미제공
